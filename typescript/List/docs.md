# Linked List    
 

*链表（Linked List）：是一种 非连续、非顺序 存储的线性结构。*  


## constructor

* `new LikedList(option)`  

```ts
option:{
  type:'oneway' | 'bothway' // 单项或双向链表  
  circle:false | true // 是否是循环链表
  storeType:'object' | 'array' | 'map' // 用什么存储
}
```  

## methods  

* `insert(item)`  
* `get(item)`   
* `delete(item)`
